CSS Zen-Garden was one of the earlier assignments I had in class. The challenge was to only use CSS and we could not touch the HTML to format and design a reputable website.

As a class we not only had to use CSS (which then was new to me) we also had to use CSS selectors based off a of a list of them.

Near the deadline I managed to finish early and had fun making adds for the website. Looking back I enjoyed the constraints this assignment brought, but I would not like to do this again.
